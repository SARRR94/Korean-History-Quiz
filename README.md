# 한능검 OX 퀴즈

한국사능력검정시험(한능검) 학습용 O/X 퀴즈 웹입니다.

## 실행 방법

1. `index.html` 파일을 더블클릭하거나
2. 브라우저 주소창에 파일 경로를 입력해 엽니다.

별도 설치나 서버 없이 바로 사용할 수 있습니다.

## 기능

- 한국사 핵심 O/X (선사·고대 / 고려 / 조선 / 근·현대 / 문화·사상)
- **내 문제집**: 퀴즈 중 ☆ 버튼으로 저장, 문제집만 따로 풀기
- 문항 범위(전체 / 내 문제집) + 시대·분야 필터, 10·20·30문항 선택
- 즉시 정오답 피드백 + 해설
- **이전 문제**: 이미 풀었던 번호까지 되돌아가 선택·해설 다시 확인 (← 버튼 또는 `←` 화살표 키)
- 오답 노트, 틀린 문제만 다시 풀기
- 최고 점수 저장 (브라우저 localStorage)
- 키보드: `O` / `X` 선택, `Enter` 다음 문제, `←` 이전 문제

## 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 화면 구조 |
| `styles.css` | 스타일 |
| `questions.js` | 문항 데이터 |
| `app.js` | 퀴즈 로직 |

문항을 추가·수정하려면 `questions.js`의 `QUIZ_QUESTIONS` 배열을 편집하세요.
